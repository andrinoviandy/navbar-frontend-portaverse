# Navigation Bar Component

A module-federated remote app for `Navbar` component in smartkmsystem apps

## Running Project
Before running the project, run `pnpm i` to install dependencies.
- Run project: `pnpm dev` (Development Mode, Non-Remote)
- Run with SSO: `pnpm dev-sso` (Development Mode, Non-Remote)
- Run as remote app: `pnpm preview` (Production Mode, Remote)
